\# Expense Tracker



A full-stack expense tracking application that helps users manage and monitor their daily expenses.



\## Tech Stack



Frontend

\- React.js

\- CSS





Backend

\- Spring Boot

\- Java

\- JWT Authentication



Database

\- MongoDB



\## Features



\- User Signup \& Login

\- Add, edit and delete expenses

\- Dashboard with charts

\- Monthly and yearly expense tracking

\- Profile management

\- PDF report generation



\## Project Structure



Expense\_Tracker

│

├── Expense-Tracker-Frontend

└── Expense-Tracker-Backend



\## How to Run



Backend

cd Expense-Tracker-Backend  

mvn spring-boot:run



Frontend

cd Expense-Tracker-Frontend  

npm install  

npm start
